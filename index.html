Sobre o Site

Este site foi feito para ajudar você a converter valores entre diferentes moedas e entender quanto cada valor vale antes de enviar dinheiro.

O conversor usa uma API de câmbio para buscar taxas atualizadas. A transferência é uma simulação que mostra quando os dados estão corretos e o valor pode ser enviado.

Use o conversor para verificar trocas entre Real, Dólar, Euro, Libra, Iene, Peso Argentino, Dólar Canadense, Dólar Australiano e Peso Mexicano.
